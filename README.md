# Kamu Pasti Bisa

Platform pembelajaran modern yang dirancang khusus untuk penguasaan materi ujian melalui metode kartu hafalan (Flipcard) dan simulasi ujian (Tes).

## Fitur Utama

- **Kartu Hafalan**: Teknik repetisi aktif untuk penguasaan konsep inti.
- **Simulasi Ujian**: Latihan soal dengan sistem penilaian otomatis.
- **Evaluasi AI**: Penilaian jawaban esai secara objektif.
- **Privasi Total**: Seluruh data progres disimpan secara lokal di peramban Anda.

## Teknologi

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Database**: Redis (untuk feedback anonim)
- **Deployment**: Deployment Publik

## Cara Penggunaan

1. Instalasi dependensi:
   ```bash
   npm install
   ```

2. Jalankan server pengembangan:
   ```bash
   npm run dev
   ```

3. Akses aplikasi melalui [http://localhost:3000](http://localhost:3000).
